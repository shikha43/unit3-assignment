## üßê How Can I Contribute?

There are several ways you can help improve this project:

### 1. üêõ Report Bugs

If you find a bug, please help us by submitting an issue to our [Issue Tracker](https://github.com/shikha43/unit3-assignment/issues).

#### Guidelines for Bug Reports:

* **Check Existing Issues:** Before submitting, please check if the bug has already been reported.
* **Clear Title:** Use a descriptive title that quickly summarizes the problem .
* **Steps to Reproduce:** Provide a clear, step-by-step description of how to reproduce the bug.
* **Expected vs. Actual Behavior:** Describe what you expected to happen and what actually occurred.
* **Environment Details:** Include relevant information like your Operating System, Browser/Version, and the project version (if applicable).

### 2. ‚ú® Suggest Features

We love new ideas! If you have a suggestion for a feature or enhancement, please open a new issue.

#### Guidelines for Feature Requests:

* **Problem:** Explain the problem this feature solves.
* **Proposal:** Describe the feature itself and how it should work.
* **Examples:** Provide examples of how users would interact with the new feature.
